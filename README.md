# Лабораторная Работа №5. Дерево Выражений  

Проект реализует дерево арифметических выражений на C++.

Дерево можно построить двумя способами:

1. Из постфиксной записи.
2. Из инфиксной записи со скобками.

Пример инфиксной записи:

(2 + 3) * 4

Она преобразуется в постфиксную:

2 3 + 4 *

После этого строится дерево выражения.

## Реализовано:

1. Построение дерева выражения из постфиксной записи.
2. Построение дерева из инфиксной записи.
3. Вычисление значения выражения.
4. Вывод выражения в инфиксной форме.
5. Вывод выражения в префиксной форме.
6. Вывод выражения в постфиксной форме.
7. Поддержка операторов +, -, *, /, ^.
8. Поддержка отрицательных чисел.
9. Определение высоты дерева.
10. Подсчёт количества операций.
11. Поддержка переменных.
12. Сворачивание константных подвыражений.


**Для хранения переменных используется собственный map на AVL-дереве.**

## Структура проекта:

example/main.cpp — примеры работы программы.
tree/tree.hpp — описание дерева выражений.
tree/tree.cpp — реализация дерева выражений.
map/AVLMap.hpp — описание AVL-map.
map/AVLMap.cpp — реализация AVL-map.
CMakeLists.txt — файл сборки.

## Сборка:

mkdir build
cd build
cmake ..
cmake --build .
